Ayuda: ¿Cómo usar la Calculadora X-tremo?

La calculadora X-tremo te permite explorar extremos relativos y absolutos de funciones multivariables de forma visual, interactiva y guiada.

Antes de usarla, es recomendable entender los conceptos clave. Para eso, podés consultar la teoría explicada paso a paso.

Según el tipo de análisis, podés activar distintos modos:

Extremos Relativos

Para analizar puntos críticos sin restricciones:

  1. Seleccioná el modo relativo desde el panel lateral.
  2. Elegí si tu función depende de una o dos variables.
  3. Ingresá la función en el campo correspondiente. Ejemplo:
    f(x, y) = x² - y²
  4. Presioná Analizar. El sistema calculará derivadas, mostrará un análisis y clasificará los puntos críticos.

    5. Este modo es ideal para funciones que no están condicionadas por restricciones externas. Si tu problema incluye condiciones, consultá el análisis absoluto con método de Lagrange.

Extremos Absolutos con Restricción

Cuando hay una condición que debe cumplirse, se aplica el método de Lagrange:

  1. Seleccioná el modo absoluto en la interfaz.
  2. Elegí entre funciones de dos o tres variables.
  3. Ingresá:
    • f(x, y, z): función objetivo. Ejemplo: x·y·z
    • g(x, y, z) y h(x, y, z): restricciones. Ejemplo: x + y + z - 32 = 0, x - y + z = 0
  4. Presioná Analizar. Se mostrarán los gradientes, el sistema de ecuaciones y la solución hallada, incluyendo los multiplicadores de Lagrange (μ y λ).

Preguntas frecuentes (FAQ)

¿Qué tipo de funciones puedo analizar?

Funciones con una, dos o tres variables, como x² + y² o x·y·z. También podés usar funciones trigonométricas, logaritmos y exponentes.

¿Cómo debo escribir las expresiones?

La ausencia de signo, implica una multiplicación (ej: xy = x·y). Agrupá con paréntesis si es necesario. Ejemplo válido: x*(y + z). No uses letras o símbolos desconocidos para el sistema, solo se trabaja con x, y, z.

¿Qué pasa si no ingreso bien la función?

La calculadora mostrará una alerta. Asegurate de revisar los operadores y de no dejar espacios vacíos o símbolos sin cerrar.

¿Dónde se ven los resultados?

Justo debajo del formulario principal. Ahí vas a ver el análisis teórico, pasos intermedios y los gráficos, si corresponde.

¿Se pueden guardar los resultados?

Por ahora no, pero podés copiar el análisis o hacer una captura. Pronto se podrá exportar como PDF o imagen.
