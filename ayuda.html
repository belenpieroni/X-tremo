Ayuda: ¿Cómo usar la Calculadora X-tremo?

Es importante conocer lo que estamos haciendo durante estos calculos, por eso es recomendable:

Ver teoría

La Calculadora X-tremo te permite encontrar extremos relativos y absolutos de funciones multivariables de manera visual, interactiva y guiada. Según el tipo de análisis que necesites, podés elegir entre distintos modos de funcionamiento:

Extremos Relativos

Para analizar puntos críticos sin restricciones:

  1. Seleccioná el modo relativo desde el panel lateral.
  2. Elegí si tu función depende de una o dos variables.
  3. Ingresá la función en el campo correspondiente. Ejemplo:
    f(x, y) = x² - y²
  4. Presioná Analizar. El sistema calculará derivadas, mostrará un análisis y clasificará los puntos críticos.

Extremos Absolutos con Restricción

Cuando hay una condición que debe cumplirse, se aplica el método de Lagrange:

  1. Seleccioná el modo absoluto en la interfaz.
  2. Elegí entre funciones de dos o tres variables.
  3. Ingresá:
    • f(x, y, z): función objetivo. Ejemplo: x·y·z
    • g(x, y, z) y h(x, y, z): restricciones. Ejemplo: x + y + z - 32 = 0, x - y + z = 0
  4. Presioná Analizar. Se mostrarán los gradientes, el sistema de ecuaciones y la solución hallada, incluyendo los multiplicadores de Lagrange (μ y λ).

Notación permitida

Podés escribir funciones usando operadores como:

  • +, -, * muy importante colocarlo siempre, / y ^ para potencias
  • sin(x), cos(y), log(z), etc.
  • Paréntesis siempre que sea necesario para agrupar
